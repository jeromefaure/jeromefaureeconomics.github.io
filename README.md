# Site GitHub Pages — Jérôme Faure

Ce dossier contient un site statique prêt à être publié sur GitHub Pages.

## Mise en ligne rapide
1. Créez un dépôt GitHub nommé `votrepseudo.github.io`
2. Déposez tous les fichiers de ce dossier à la racine du dépôt
3. Activez GitHub Pages dans **Settings > Pages**
4. Choisissez la branche `main` et le dossier `/ (root)`

## Fichiers
- `index.html` : page d’accueil
- `assets/style.css` : styles
- `pages/` : pages par rubrique du CV
